## Disposables.Create

`using` statements nicely cleanup on exit if you have a `IDisposable` or you can create a simple disposable object by passing in a lambda.  

<!-- snippet: disposables -->
<a id='snippet-disposables'></a>
```cs
using (Disposables.Create(() => callCount++))
{
    //code
}
```
